# MailFlow AI

A modern, web-based interface for your email marketing software.

## Features
- **Dashboard**: Real-time progress monitoring, logs, and controls.
- **Settings**: Manage SMTP configurations easily.
- **Template Editor**: Edit your HTML email template directly in the browser.
- **Recipients**: View and upload your mailing list (CSV).

## How to Run
Double-click `start_app.bat` to launch the system.

## Manual Start
1. **Backend**:
   ```bash
   pip install -r requirements.txt
   python server.py
   ```
2. **Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
